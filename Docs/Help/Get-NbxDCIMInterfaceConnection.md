# Get-NbxDCIMInterfaceConnection

## SYNOPSIS

Retrieves interface connection objects (links between interfaces) from NetBox.

## SYNTAX

### __AllParameterSets

```
Get-NbxDCIMInterfaceConnection [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### Example 1

PS C:\> Get-NbxDCIMInterfaceConnection -ID 9
