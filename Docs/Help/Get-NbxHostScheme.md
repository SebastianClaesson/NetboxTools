# Get-NbxHostScheme

## SYNOPSIS

Retrieves host scheme definitions from NetBox (URI/host naming schemes, etc.).

## SYNTAX

### __AllParameterSets

```
Get-NbxHostScheme [<CommonParameters>]
```

## DESCRIPTION

Gets host scheme objects from NetBox. Use `-Id` to request specific schemes.
